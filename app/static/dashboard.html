How It Works

This system uses an XGBoost machine learning model trained on 1M+ transaction records from the Online Retail II dataset. It analyzes 30 customer features including purchase frequency, monetary value, recency, temporal trends, basket behavior, and lifecycle patterns.

Model Performance: The model achieves 99.3% accuracy (R²) with a mean absolute error of just £170. This means on average, predictions are within ±£170 of actual future revenue.

Confidence Intervals: Each prediction includes a confidence range based on the model's historical performance, helping you understand prediction uncertainty.
